Detailed Description

template<typename TItemTraits, typename THashTraits = HashTraits<typename TItemTraits::Key>, typename TSettings = HashSetSettings>
class momo::HashSetCore< TItemTraits, THashTraits, TSettings >

All HashSetCore functions and constructors have strong exception safety, but not the following cases:

  1. Functions Insert receiving many items have basic exception safety.
  2. Function Remove receiving predicate has basic exception safety.
  3. Functions MergeFrom and MergeTo have basic exception safety.
